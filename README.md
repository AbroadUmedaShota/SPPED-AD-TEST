# SPEED AD - Exhibition & Seminar Lead Management Solution

## Overview
SPEED AD is an all-in-one web application designed to streamline the lead management process for exhibitions and seminars. It integrates business card digitization, web surveys, thank-you emails, and data analysis into a single platform.

## Key Features
- **Smart Survey Builder**: Create and manage web surveys with various question types, including handwriting.
- **Business Card Digitization**: High-precision digitization combining OCR and manual verification.
- **Automated Follow-up**: Send thank-you emails automatically based on survey responses.
- **Analytics Dashboard**: Visualize collected data with interactive charts and export results to Excel/CSV.
- **Admin Console**: Manage users, operators, and data entry workflows.

## Project Structure
- `02_dashboard/`: User-facing marketing dashboard and survey management.
- `03_admin/`: Backend operations console for data entry and operator management.
- `04_first-login/`: Onboarding tutorial for new users.
- `data/`: Mock JSON datasets for the frontend.
- `docs/`: Comprehensive project documentation.
