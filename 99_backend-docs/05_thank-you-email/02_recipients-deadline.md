# 送信対象と期限

## 送信対象の考え方

| 条件 | 扱い |
| --- | --- |
| アンケート回答がある | 送信候補の基礎 |
| メールアドレスがある | 送信可能性の基礎 |
| 名刺データ化が完了している | 対象者確定の前提になりやすい |
| 送信除外条件に該当する | 送信対象から除外 |
| すでに送信済み | 再送禁止または再送条件確認 |

## 期限の考え方

既存資料では、御礼メールが一斉配信されるまでの期間を営業日で扱い、会期の最終日を0営業日目として計算する考え方が確認されています。

ただし、本番の具体的な送信期限、期限延長、例外対応はサーバー側仕様として確定が必要です。

## バックエンドで必要な判定

- 会期終了日
- 名刺データ化完了日
- 送信期限
- 送信対象者数
- 送信済み件数
- 送信不可理由
- 期限切れ状態

## 未確認事項

- 送信期限を営業日で統一するか、プランや契約で変わるか。
- 送信対象を名刺データ化済み連絡先に限定するか、メールアドレス回答者全体を含めるか。
- 期限切れ後の例外送信を許可するか。
