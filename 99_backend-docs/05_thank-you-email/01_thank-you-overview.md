# お礼メールの位置づけ

## 概要

お礼メールは、アンケート回答者に対して会期後のフォロー連絡を行う主要機能です。

SPEED ADでは、名刺データ化によって整った連絡先情報と、アンケート回答内容をもとに送信対象を確認し、利用者が送信を実行する流れとして整理します。

## 対象範囲

| 対象 | 内容 |
| --- | --- |
| 設定 | 送信するか、件名、本文、差出人、送信方法 |
| 対象者 | 回答者、名刺データ化済み連絡先、送信除外 |
| 期限 | 会期終了日や営業日を起点にした送信可能期間 |
| 実行 | 送信確認、送信実行、結果確認 |
| 失敗系 | 期限切れ、再送禁止、部分失敗、送信不可 |

## システムメールとの違い

- お礼メール本文そのものは回答者向けのフォローメールです。
- `01_system-mail` で整理している「御礼メール送信依頼」は、利用者に送信実行を促すシステム通知です。
