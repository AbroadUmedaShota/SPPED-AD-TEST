# 進捗・納品・通知

## 進捗フェーズ

| フェーズ | 内容 |
| --- | --- |
| 受付 | 名刺データ化申込が保存された状態 |
| 回収 | 会期中に名刺画像や手入力情報が集まる状態 |
| 入力 | OCR、手入力、オペレーター入力が進む状態 |
| 照合 | 入力結果の一致、不一致、エスカレーションを確認する状態 |
| CSV作成 | 納品用CSVへ反映する状態 |
| 納品 | データ化完了として利用者が取得できる状態 |

## 通知との接続

| 接続 | 内容 |
| --- | --- |
| 名刺データ化申込通知 | 設定保存・申込時の受付通知 |
| 想定件数到達通知 | 想定件数の80%または100%到達時の注意喚起 |
| 名刺データ化完了通知 | 完了CSVやダウンロードを促す通知 |
| 御礼メール送信依頼 | データ化完了後、送信対象確認・送信実行を促す通知 |

## 未確認事項

- 完了判定を「予定日」「CSVアップロード」「完了件数」のどれで確定するか。
- 完了通知の送信済み管理と再送ルール。
- データ化完了後の差し戻し、再アップロード、請求補正の扱い。
