# 名刺データ化の位置づけ

## 目的

名刺データ化は、アンケート回答者に紐づく名刺画像を、営業フォローやCSV出力で使える連絡先データへ整える工程です。

SPEED ADでは、アンケート回答、名刺画像、データ化結果、お礼メール、請求が同じアンケート案件に接続します。

## 主な入力と出力

| 区分 | 内容 |
| --- | --- |
| 入力 | アンケート回答、名刺画像、手入力情報、データ化設定 |
| 中間処理 | OCR、オペレーター入力、照合、エスカレーション |
| 出力 | 名刺データ化完了CSV、名刺画像ZIP、回答+名刺データ |
| 接続先 | お礼メール送信対象、請求明細、SPEEDレビュー |

## バックエンドで必要な管理

- 名刺画像の登録有無
- データ化対象かどうか
- 依頼枚数、完了件数、完了予定日
- 入力・照合・CSVアップロードの進捗
- ダウンロード期限
- 請求対象件数

- オペレーター割当の本番ルール
