# バッチ処理一覧

## 旧Wikiで確認できる代表処理

| 処理 | 時刻 | 内容 |
| --- | --- | --- |
| データ化完了予定日のステータス更新 | 毎日10:00 | アップ待ちからアップ完了へ変更するとされる |
| 会期開始ステータス更新 | 毎日0:00 | 開始日が当日のアンケートを会期中へ変更するとされる |
| 会期前日・利用開始日・DL期限通知 | 毎日0:00 | 条件に当てはまるアンケートへメール送信するとされる |
| 会期終了通知 | 毎日23:59 | 利用期間が本日までのアンケートへ終了メールを送るとされる |
| 名刺データ化完了通知 | 毎日0:10 | 完了予定日かつアップ完了のアンケートへ通知するとされる |
| 御礼メール一括送信依頼 | 毎日0:10 | データ化完了予定日から一定期間後に送信依頼するとされる |
| 請求書作成・メール送信 | 月末23:58 | 月末請求を作成し通知するとされる |
| 管理画面用実績集計 | 毎時0分 | 実績集計処理とされる |

## 扱い

- 上記は旧Wikiベースの整理です。
- 実ジョブ名、ジョブ基盤、実行ログ、失敗時リトライは未確認です。
- 正式化する場合は、バッチ処理を「状態更新」「通知」「請求」「集計」に分けて仕様化する必要があります。
