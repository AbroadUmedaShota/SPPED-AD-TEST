目視比較による観測メモです。「本番に不足」はモックにあって本番にない要素、「本番先行」は本番にあってモックにない要素（モック未更新の可能性）、「UI差」はレイアウト・表示の違い、「要実機確認」は静止画では判定できない挙動です。差分の解消可否・優先度・対応方針は別途判断します。 スクショは アーカイブ/画面整理用フォルダ/ 取り込み分と、本番をPlaywrightで撮影した分（読み取り専用）を images/ に格納しています。
