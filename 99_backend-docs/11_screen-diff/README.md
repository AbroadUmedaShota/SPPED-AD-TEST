# 11_screen-diff / 画面差分（モックアップ↔本番）

モックアップと本番環境を画面ごとにスクリーンショットで並べ、**本番に足りない・異なる点**を洗い出すための整理ページです。

- 左 = モックアップ（`abroadumedashota.github.io/SPPED-AD-TEST/`）
- 右 = 本番（`speed-ad.com`）
- 差分の見方 = モックアップにはあって本番にはない、または表示が異なる点

正本は [`index.html`](./index.html)。画面ごとのスクショは [`images/`](./images/) に格納しています（元データは `アーカイブ/画面整理用フォルダ/` から取り込み、ファイル名を整理済み）。
