# サービス概要

## 基本説明

SPEED AD は、展示会やセミナーで発生するアンケート回収、名刺取得、名刺データ化、会期後フォローをひとつの流れで扱う BtoB 向け業務改善サービスです。

来場者は QR コードなどからアンケートに回答し、必要に応じて名刺画像や連絡先情報を登録します。利用者は回答データと名刺情報を同じアンケート案件の中で確認し、会期後の営業フォロー、CSV出力、請求確認へ進みます。

## サービスが解く課題

| 課題 | SPEED ADで扱うこと |
| --- | --- |
| アンケートと名刺が別管理になる | 回答、名刺、メモを同じアンケート案件に紐づける |
| 会期後のフォローが遅れる | 名刺データ化、レビュー、CSV出力、お礼メールを同じ流れで扱う |
| 請求・データ化・メールが分断される | 名刺件数、送信対象、請求明細をアンケート案件単位で整理する |
| 運用状態が分かりづらい | 会期、データ化、DL期限、請求などの状態を見える化する |

## 主要機能

- WEBアンケートの作成、回答受付、回答データ管理
- 名刺撮影、名刺画像登録、名刺データ化依頼
- 名刺データ化完了後のCSVダウンロード
- お礼メール設定、送信対象確認、送信実行
- グループ管理、メンバー招待、共同利用
- 請求書確認、請求通知、請求明細確認
- 管理側のアンケート、請求、営業日、オペレーター、実績管理

## バックエンド整理で見るべき観点

| 観点 | 内容 |
| --- | --- |
| 状態 | 会期前、会期中、会期終了、データ化中、データ化完了、DL期限終了 |
| 時間起点 | 会期開始日、会期終了日、データ化完了予定日、月末、DL期限 |
| 操作起点 | アカウント作成、パスワード再設定、グループ招待、名刺データ化申込 |
| 通知 | 会期前日、会期開始、会期終了、データ化完了、請求、送信依頼 |
| 計算 | 名刺データ化見積、送信対象件数、請求明細、営業日、保存期間 |
