# SM-002 パスワード再設定時

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | アカウント |
| Excel件名 | `【SPEED AD】パスワード再発行のご案内` |
| Gmail件名 | `【SPEED AD】パスワード再発行` |
| 対象 | パスワード再設定を依頼したユーザー |
| ソース状態 | Excel・Gmail実配信確認、差分あり |

## 送信タイミング

随時。

## 条件

パスワード再設定の依頼操作が行われた時。

## Excel版

```text
件名:
【SPEED AD】パスワード再発行のご案内

本文:
いつもSPEED ADをご利用いただきまして誠にありがとうございます。

パスワードの再設定は、下記URLをクリックして設定画面より行ってください。
※URLの有効期限は●分です。

[URL]
```

## Gmail実配信版

```text
件名:
【SPEED AD】パスワード再発行

本文:
この度はSPEED AD（スピードアド）にアクセス頂き、誠にありがとうございます。

以下のURLからパスワードの再設定をお願いいたします。
[パスワード再設定URL]

[共通フッター]
```

## 差分メモ

- 件名が `パスワード再発行のご案内` から `パスワード再発行` に短縮されている。
- Excel版にはURL有効期限の注記があるが、Gmail実配信例では確認できない。
- 冒頭文もExcel版とGmail実配信版で異なる。

## 未確認事項

- 現行仕様としてURL有効期限をユーザーに表示するか。
- パスワード再設定URLの有効時間。
