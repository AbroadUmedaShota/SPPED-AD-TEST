# SM-013 アカウント作成依頼メール（新規ユーザー）

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | グループ |
| 件名 | `【SPEED AD】アカウント作成のご案内` |
| 対象 | 招待先メールアドレスがユーザーDBに存在しないユーザー |
| ソース状態 | Excel・Gmail実配信確認 |

## 送信タイミング

随時。

## 条件

グループ招待の送信操作が行われ、該当メールアドレスがユーザーDBに存在しなかった時。

## Excel版

```text
件名:
【SPEED AD】アカウント作成のご案内

本文:
[メールアドレス]　様

いつもSPEED ADをご利用いただきまして誠にありがとうございます。

お客様宛に、グループ「グループ名」への招待が届きました。

SPEED ADのご利用アカウントを作成後にご利用いただけますので、
以下のURLからSPEED ADアカウント作成をお願いします。
[アカウント作成画面のURL]

※アカウント作成後に再度グループ招待メールが届きますので、メールに記載のリンクから、参加承認をお願いいたします。

ご不明な点などがございましたら
お手数ですが下記までご連絡をお願いいたします。
```

## 差分メモ

- Gmail実配信はExcel本文に共通フッターが付く形。
- 宛名は氏名ではなく招待先メールアドレスになる。

## 未確認事項

- アカウント作成URLの有効期限。
- アカウント作成完了後に再送されるグループ招待メールの自動送信条件。
