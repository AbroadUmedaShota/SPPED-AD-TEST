# SM-003 名刺データ化申込時

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | 名刺データ化 |
| 件名 | `【SPEED AD】「[アンケート名]」の名刺データ化のお申込み、ありがとうございます。` |
| 対象 | アンケート作成者、または対象グループの通知先 |
| ソース状態 | Excel・Gmail実配信確認 |

## 送信タイミング

随時。

## 条件

有料となるスタンダードプラン以上で、名刺データ化設定の申し込みまたは保存がされた時。

## 差分メモ

- Gmail実配信は、Excel本文に共通締め文と共通フッターが付く形。
- 宛名は個人名だけでなくグループ名相当が入る例があるため、本資料では `[通知先名]` とする。

## 未確認事項

- `御礼メール設定` の表示値一覧。
- 申し込み後に保存変更された場合の再送有無。
