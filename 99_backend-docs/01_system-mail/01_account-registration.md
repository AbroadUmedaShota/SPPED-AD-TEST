# SM-001 新規アカウント作成時（配信なし）

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | アカウント |
| 件名 | 配信なし |
| 対象 | なし |
| ソース状態 | 要件変更により現行配信なし |

## 送信タイミング

配信なし。

## 条件

要件変更により、新規アカウント作成時のシステムメール配信は行いません。

過去資料では、メールアドレス登録後にパスワード設定URLを送信する想定が確認されていますが、現行仕様としては配信対象外です。

## Excel版

過去テンプレートとして確認された内容です。現行配信文面ではありません。

```text
件名:
【SPEED AD】ご登録メールアドレスの確認

本文:
この度はSPEED AD（スピードアド）にアクセス頂き、誠にありがとうございます。

以下のURLからパスワードを設定し、ご登録の完了をお願いいたします。

[URL]
```

## Gmail実配信版

過去の実配信例として確認された内容です。現行配信文面ではありません。

```text
件名:
【SPEED AD】ご登録メールアドレスの確認

本文:
この度はSPEED AD（スピードアド）にアクセス頂き、誠にありがとうございます。

以下のURLからパスワードを設定し、ご登録の完了をお願いいたします。

[パスワード設定URL]

今後ともSPEED ADをよろしくお願い申し上げます。

[共通フッター]
```

## 差分メモ

- 要件変更により、現行仕様では新規アカウント作成時メールは配信しない。
- Gmail実配信では本文末尾に `今後ともSPEED ADをよろしくお願い申し上げます。` と共通フッターが付く。
- URLは実配信ではパスワード設定用トークンURLだが、本資料ではプレースホルダー化する。
