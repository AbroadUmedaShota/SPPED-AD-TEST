# SM-001 新規アカウント作成時

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | アカウント |
| 件名 | `【SPEED AD】ご登録メールアドレスの確認` |
| 対象 | 新規登録メールアドレス |
| ソース状態 | Excel・Gmail実配信確認 |

## 送信タイミング

随時。

## 条件

新規アカウント作成で、メールアドレス登録ボタンが押された後。

## Excel版

```text
件名:
【SPEED AD】ご登録メールアドレスの確認

本文:
この度はSPEED AD（スピードアド）にアクセス頂き、誠にありがとうございます。

以下のURLからパスワードを設定し、ご登録の完了をお願いいたします。

[URL]
```

## Gmail実配信版

```text
件名:
【SPEED AD】ご登録メールアドレスの確認

本文:
この度はSPEED AD（スピードアド）にアクセス頂き、誠にありがとうございます。

以下のURLからパスワードを設定し、ご登録の完了をお願いいたします。

[パスワード設定URL]

今後ともSPEED ADをよろしくお願い申し上げます。

[共通フッター]
```

## 差分メモ

- Gmail実配信では本文末尾に `今後ともSPEED ADをよろしくお願い申し上げます。` と共通フッターが付く。
- URLは実配信ではパスワード設定用トークンURLだが、本資料ではプレースホルダー化する。

## 未確認事項

- トークン有効期限の表示有無。
- 未登録メールアドレスがグループ招待から作成される場合との文面差分。
