# SM-012 グループ招待メール（既存ユーザー）

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | グループ |
| 件名 | `【SPEED AD】グループ「[グループ名]」招待のご案内` |
| 対象 | 招待先メールアドレスがユーザーDBに存在するユーザー |
| ソース状態 | Excel・Gmail実配信確認 |

## 送信タイミング

随時。

## 条件

グループ招待の送信操作が行われ、該当メールアドレスがユーザーDBに存在した時。

## Excel版

```text
件名:
【SPEED AD】グループ「グループ名」招待のご案内

本文:
[アカウント氏名]　様

いつもSPEED ADをご利用いただきまして誠にありがとうございます。

お客様宛に、グループ「グループ名」への招待が届きました。
以下のURLから参加承認をお願いします。
[グループ参加承認のURL]

※既にSPEED ADのアカウントをお持ちの方は、ログイン後に参加承認ができます。

ご不明な点などがございましたら
お手数ですが下記までご連絡をお願いいたします。
```

## Gmail実配信版

```text
件名:
【SPEED AD】グループ「[グループ名]」招待のご案内

ご不明な点などがございましたら
```

## 差分メモ

- Excel版は `グループ名` と固定文字のように記載されているが、Gmail実配信ではグループ名が差し込まれる。
- Gmail実配信はExcel本文に共通フッターが付く形。

## 未確認事項

- 招待承認URLの有効期限。
- 既存ユーザーが未ログイン状態でリンクを開いた場合の遷移。
