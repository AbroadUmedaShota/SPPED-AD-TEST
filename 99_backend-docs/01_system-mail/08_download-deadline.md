# SM-008 ダウンロード期限

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | 期限通知 |
| 件名（1週間前） | `【SPEED AD】「[アンケート名]」のダウンロード期限について（1週間前）` |
| 件名（本日まで） | `【SPEED AD】「[アンケート名]」のダウンロード期限について（本日まで）` |
| 対象 | ダウンロード期限を迎えるアンケートの作成者 |
| ソース状態 | Excelのみ確認 |

## 送信タイミング

- 1週間前: 毎日AM10:00。
- 24時間前: AM0:00。

## 条件

- 1週間前: ダウンロード期限7日前のアンケート。
- 24時間前: ダウンロード期限24時間前、期限当日のアンケート。
