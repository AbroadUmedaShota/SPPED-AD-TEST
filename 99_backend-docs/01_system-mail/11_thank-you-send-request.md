# SM-011 御礼メール送信依頼

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | 御礼メール |
| 件名 | `【SPEED AD】「[アンケート名]」の御礼メール一括送信のお願い` |
| 対象 | 御礼メール一括送信の準備が整ったアンケートの作成者 |
| ソース状態 | Excel・Gmail実配信確認 |

## 送信タイミング

随時。Excel版ではデータ化完了メールの直後。

## 条件

データ化完了メールの直後。完了予定日前の場合は完了予定日のAM10:00、完了予定日のAM10:00以降の場合はデータアップ完了時に随時送信。

## Excel版

```text
件名:
【SPEED AD】「[アンケート名]」の御礼メール一括送信のお願い

本文:

つきましては、以下URLの御礼メール設定から内容をご確認の上、
「メール送信」ボタンのクリックをお願い致します。

[御礼メール設定のURL]

御礼メール送信の期限
[御礼メール送信の期限（yyyy年mm月dd日（曜日）23時59分）]

※お客様が「メール送信」ボタンをクリックするまで、御礼メールの一括送信は行われません。
※御礼メール送信の期限は1週間です。
　期限を過ぎると御礼メールの一括送信ができなくなりますのでご注意ください。

ご不明な点などがございましたら
お手数ですが下記までご連絡をお願いいたします。
```

## 差分メモ

- Gmail実配信はExcel本文に共通フッターが付く形。
- これは御礼メール本文そのものではなく、御礼メール送信操作を促すシステム通知として扱う。
