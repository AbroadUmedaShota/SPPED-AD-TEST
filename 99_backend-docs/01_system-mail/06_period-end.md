# SM-006 会期終了

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | アンケート会期 |
| 件名 | `【SPEED AD】「[アンケート名]」のご利用期間が終了しました` |
| 対象 | 会期終了日を迎えるアンケートの作成者 |
| ソース状態 | Excel・Gmail実配信確認 |

## 送信タイミング

PM23:59。

## 条件

会期終了日を迎えるアンケートの作成者に対して送信。名刺データ化設定と名刺画像の有無により本文の一部が分岐する。

## Excel版

### 名刺入力あり

```text
件名:
【SPEED AD】「[アンケート名]」のご利用期間が終了しました

本文:
[アカウント氏名]　様

いつもSPEED ADをご利用いただきまして誠にありがとうございます。

お客様が作成された「[アンケート名]」のご利用期間が終了しました。
獲得されたアンケート回答件数は[件数]件でした。

これより収集された名刺データの状況を確認し、名刺データ化を進めて参ります。
名刺データ化の完了予定日は、[完了予定日]の見込です。

また収集されたアンケートの回答、及び名刺画像は、
SPEED ADにログイン後、アンケート一覧からダウンロードが可能です。

なお、これ以降「[アンケート名]」では新たな回答の送信ができませんので、
再度ご利用になられたい場合は新しいアンケートの作成をお願いいたします。
```

### 名刺入力なし

```text
件名:
【SPEED AD】「[アンケート名]」のご利用期間が終了しました

本文:
[アカウント氏名]　様

いつもSPEED ADをご利用いただきまして誠にありがとうございます。

お客様が作成された「[アンケート名]」のご利用期間が終了しました。
獲得されたアンケート回答件数は[件数]件でした。
※このアンケートからはデータ化対象の名刺がないため、名刺データ化は行われません。

収集されたアンケートの回答、及び名刺画像は、
SPEED ADにログイン後、アンケート一覧からダウンロードが可能です。

なお、これ以降「[アンケート名]」では新たな回答の送信ができませんので、
再度ご利用になられたい場合は新しいアンケートの作成をお願いいたします。
```

## 差分メモ

- Gmail実配信では、名刺入力あり/なしの両方に相当する本文が確認された。
- Gmail実配信はExcel本文に共通フッターが付く形。

## 未確認事項

- 回答件数が0件の場合の表示文言。
- 名刺画像はあるがデータ化対象外となった場合の分岐条件。
