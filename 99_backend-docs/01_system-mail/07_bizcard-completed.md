# SM-007 名刺データ化完了

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | 名刺データ化 |
| 件名 | `【SPEED AD】「[アンケート名]」の名刺データ化完了のお知らせ` |
| 対象 | 名刺データ化が完了したアンケートの作成者 |
| ソース状態 | Excel・Gmail実配信確認 |

## 送信タイミング

AM10:00または随時。

## 条件

完了予定日前の場合、完了予定日のAM10:00に送信。完了予定日のAM10:00以降の場合、データアップ完了時に随時送信。
