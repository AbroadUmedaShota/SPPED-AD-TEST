# SM-014 共通フッター

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | 共通 |
| 対象 | SPEED ADシステムメール共通の署名・問い合わせ案内 |
| ソース状態 | Excel・Gmail実配信確認、既存補助テンプレート差分あり |

## 送信タイミング

各システムメール本文末尾に付与。

## 条件

個別メールテンプレートが共通フッターを使用する場合。

## 差分メモ

- Excel版はフッターの前に `今後ともSPEED ADをよろしくお願い申し上げます。` を含むが、Gmail実配信では個別本文側に付くものと付かないものがある。
- 既存補助テンプレート `docs/テンプレート/email/invoice-notification-email-template_jp.txt` は住所表記が異なるため、現行正本の確認が必要。
- Excelの送信先設定には内部アーカイブ用のBCCがあるが、shared資料では具体アドレスを記載しない。
