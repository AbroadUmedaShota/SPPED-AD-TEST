# SM-004 会期前日

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | アンケート会期 |
| 件名 | `【SPEED AD】「[アンケート名]」の会期前日となりました` |
| 対象 | 展示会会期開始日の前日を迎えるアンケートの作成者 |
| ソース状態 | Excel・Gmail実配信確認、置換漏れ注意 |

## 送信タイミング

AM0:00。

## 条件

展示会会期の開始日の前日AM0:00に、その展示会アンケートの作成者に対して送信。

## 差分メモ

- Gmail実配信例で、設定状況見出しの `「[アンケート名]」の設定状況` が置換されずに残っているものが確認された。
- 件名と本文冒頭のアンケート名は置換されていたため、見出し部分だけの置換漏れの可能性がある。
