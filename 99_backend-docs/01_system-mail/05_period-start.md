# SM-005 会期開始

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | アンケート会期 |
| 件名 | `【SPEED AD】「[アンケート名]」のご利用日を迎えました` |
| 対象 | 展示会会期開始日を迎えるアンケートの作成者 |
| ソース状態 | Excel・Gmail実配信確認 |

## 送信タイミング

AM0:00。

## 条件

展示会会期の開始日AM0:00に、その展示会アンケートの作成者に対して送信。

## Excel版

```text
件名:
【SPEED AD】「[アンケート名]」のご利用日を迎えました

本文:
[アカウント氏名]　様

いつもSPEED ADをご利用いただきまして誠にありがとうございます。

お客様が作成された「[アンケート名]」のご利用開始日を迎えました。

これから会期終了日までの間に回答されたアンケートは有効回答として集計されます。
また回答と合わせて添付された名刺画像は、名刺データ化が行われます。
ぜひ多くのリード獲得に向けて、SPEED ADをご活用ください。

なお、これ以降は以下の変更ができません。
緊急でご入用の際は、直接お問合せください。

・アンケート名や展示会会期の変更。
・アンケートの設問の変更、またアンケートの削除。
・名刺データ化設定の変更。
```
