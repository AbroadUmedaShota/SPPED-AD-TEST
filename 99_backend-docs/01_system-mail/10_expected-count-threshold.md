# SM-010 想定件数到達

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | 名刺データ化 |
| 件名（80%） | `【SPEED AD】想定件数80％到達のご案内` |
| 件名（100%） | `【SPEED AD】想定件数100％到達のご案内` |
| 対象 | 会期中の名刺画像付き回答件数が閾値に達したアンケートの作成者 |
| ソース状態 | Excel・Gmail実配信確認 |

## 送信タイミング

随時。

## 条件

- 80%: 会期開始後の名刺画像付きアンケート回答件数が、想定件数の80%を超えた時。
- 100%: 会期開始後の名刺画像付きアンケート回答件数が、想定件数の100%を超えた時。
