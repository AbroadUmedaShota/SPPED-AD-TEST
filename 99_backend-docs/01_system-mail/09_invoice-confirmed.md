# SM-009 請求確定・請求書送付

## 基本情報

| 項目 | 内容 |
| --- | --- |
| カテゴリ | 請求 |
| 件名 | `【SPEED AD】yyyy年mm月分のご請求書の送付` |
| 対象 | 請求対象ユーザーまたは請求先 |
| ソース状態 | Excel・Gmail実配信確認、既存補助テンプレート差分あり |

## 送信タイミング

Excel版では随時。バッチ資料・Gmail実配信の観測上は月次請求メールとして扱う。

## 条件

Excel版では請求確定ボタン押下時。運用上は月次請求確定時に請求書URLとPDF添付を送付する。

## 差分メモ

- 既存補助テンプレート `docs/テンプレート/email/invoice-notification-email-template_jp.txt` は件名が `ご請求金額のお知らせ` で、税込表記・住所表記も本資料の観測値と異なる。
- Excel版は `請求確定ボタン押下時`、Gmail実配信は月次送付メールとして確認された。
- Gmail実配信ではPDF添付がある。
