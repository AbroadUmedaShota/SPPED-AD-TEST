# ダウンロード期限

## 基本整理

ダウンロード期限は、アンケート回答データ、集計データ、名刺画像、名刺データ化完了データの取得可否に関わる期限です。

既存資料では、データ保存期間は会期最終日を0日目として計算する考え方が確認されています。現行モックでは、明示期限がない場合に会期終了日から一定期間後をダウンロード期限として扱う実装があります。

## バックエンドで必要な情報

| 情報 | 用途 |
| --- | --- |
| 会期終了日 | 期限計算の起点 |
| 明示DL期限 | 例外的に設定された期限がある場合の優先値 |
| データ化完了日 | 名刺データ取得開始の目安 |
| 期限超過フラグ | 取得可否、通知、一覧表示の制御 |
| 期限例外 | 個別延長や特別対応の管理 |

## 通知との関係

| 通知 | 目的 | 状態 |
| --- | --- | --- |
| DL期限7日前 | 期限前に取得を促す | Excelのみ確認 |
| DL期限24時間前または当日 | 期限直前の取得を促す | Excelのみ確認 |

## 未確認事項

- 本番の保存期間、個別延長、期限例外の管理方法は未確認です。
- Gmail実配信としてDL期限メールは確認できていません。
- 期限計算を営業日で扱うか暦日で扱うかは、対象データ種別ごとに確認が必要です。
