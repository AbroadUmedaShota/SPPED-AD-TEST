アンケート作成画面の UI をリニューアルしました

アンケート作成画面の UI を全面的に刷新し、複雑な設問構成も操作の手数を抑えて組み立てられるよう、編集パネルとプレビューの両面を再設計しました。既存のアンケートはそのまま引き続きご利用いただけます。

主な変更点

編集画面を「設問リスト・編集パネル・プレビュー」の3カラム構成に変更しました。これまで設問の構造把握とプレビュー確認のために画面を切り替えていた工程が、1画面で完結します。

  • ドラッグ並べ替え：設問カードを上下にドラッグするだけで順番を変更可能。グループ内・グループ間の移動もそのまま行えます。
  • 分岐ロジックの可視化：「この設問でAを選んだら次は問5に飛ばす」といった分岐条件を、設問カード上に小さなフラグとして表示します。
  • プレビューの即時反映：右ペインのプレビューが、編集内容に応じてリアルタイムに更新されます。設問の選択肢や順番、必須・任意の切り替えがすぐに目で見て確認できます。
  • テンプレート切り替え：展示会向け・カンファレンス向け・社内アンケート向けなどの設問パターンを、ワンクリックで切り替えられます。

既存アンケートへの影響

すでに作成・公開済みのアンケートは、UI 切り替え後もそのまま動作します。回答 URL・回答データの構造・CSV ダウンロードの列順は変更ありません。

ただし以下の点にご注意ください。

  • 編集画面で「下書き保存」していた状態は、新UI で開いた際に下書きを引き継ぎますが、保存形式の差異により一部設問のスタイル指定（色・余白など）がデフォルト値に戻る場合があります。
  • 旧UI に戻すオプションは、移行期間の2026年7月末まで設定画面から切り替えられます。

操作ガイド

主要な操作手順は、ヘルプセンターの「アンケート作成（新UI）」セクションにまとめています。動画ガイド（各3〜5分）もあわせてご用意していますので、初めて新UI に触れる際にご活用ください。

フィードバックのお願い

UI 改善は、お客様からのご意見を継続的に反映していく予定です。お気づきの点や使いづらい箇所がありましたら、画面右下のフィードバックボタンよりお気軽にお寄せください。寄せられた声は週次で開発チームに共有しています。
