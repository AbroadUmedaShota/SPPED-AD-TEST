グラフ画面に CSV / PNG の一括エクスポートを追加しました

表示中のグラフをそのまま外部資料に持ち込めるよう、グラフ画面に CSV と PNG の同時ダウンロードを追加しました。展示会後の社内共有資料の作成手数を、出力→整形→添付の3工程から1工程に短縮できます。

追加した機能

グラフ画面の右上に「エクスポート」ボタンを追加し、ワンクリックで以下のファイルをまとめてダウンロードできるようにしました。

  • CSV：表示中のグラフに使われている数値データを、設問名・選択肢名・回答数の列構成で出力します。Excel・Google スプレッドシートでそのまま開いて加工できます。
  • PNG：実際の見え方を保ったままの高解像度画像（2倍密度）。報告書のキャプチャ画像として、画質を落とさずに貼り付けられます。
  • ZIP：CSV と PNG を同じファイル名で1つの ZIP に同梱。社内共有時の取り違えを防げます。

想定する使いどころ

これまで「画面キャプチャを撮り、Excel で別途集計し直して添付する」という流れで作っていた資料を、以下のような場面で時間短縮できます。

  • 展示会終了直後の社内共有レポート（当日中の初動共有）
  • 営業・マーケティング会議の振り返り資料
  • 経営層・関係部門向けの報告ドキュメント
  • 顧客企業との会期後ミーティングでの提示資料

ご利用方法

サイドメニューの「グラフ」から任意の集計を開き、右上の「エクスポート」ボタンを押すと、CSV / PNG / ZIP の選択肢が表示されます。出力対象は現在画面に表示中のグラフ範囲（フィルタ・期間・対象アンケートが反映された状態）です。

絞り込み条件を変更したい場合は、サイドパネルでフィルタを調整してから再度エクスポートしてください。出力に時間がかかる大規模データでは、進行状況をモーダルに表示します。

対象プラン・権限

すべてのプランで利用できます。プロジェクトのメンバー権限が「閲覧のみ」のアカウントもエクスポート可能です。管理者による個別の機能解放手続きは不要です。
