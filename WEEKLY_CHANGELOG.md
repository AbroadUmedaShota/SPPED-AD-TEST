# 変更履歴 (2025-10-08 〜 2026-02-07)

## 2026-02-07 (土)
- **サンプル回答用画像のリアリスティック化と命名規則の変更**:
    - `media/generated/sv_0001_26009/` 配下のサンプル画像を、同一画像の複製から動的に生成された多様な内容の画像に更新。
    - 命名規則を `アンケートID_連番_判別コード`（例: `sv_0001_26009_0001_1.jpg`）に統一。
    - 画像生成用のPythonスクリプト `tools/generate_realistic_images.py` を追加。
- **詳細分析画面の機能強化と描画エンジン刷新**:
    - **ApexChartsへの完全移行**: 描画ライブラリをChart.jsからApexCharts（SVGベース）に変更。
    - **グラフ種類の動的切り替え**: 棒グラフと円グラフを瞬時に切り替えられる機能を実装。
    - **凡例のインテリジェント配置**: ラベルの長さや項目数に基づき、凡例を右側または下部に自動配置。
    - **Excelレポート出力機能の実装**: `ExcelJS`を使用し、集計データとグラフ画像をセットにした詳細レポートの一括出力機能を実装。
