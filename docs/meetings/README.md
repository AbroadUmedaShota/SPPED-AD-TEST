# 会議用資料（Meeting Packets）

会議で使う「配布資料・アジェンダ・決定事項」を日付フォルダにまとめます。

- 会議メモ（議事録・走り書き）は `docs/notes/meetings/` を優先
- ここは「当日配布して参照するための資料（パケット）」置き場

## ルール（推奨）
- フォルダ: `YYYY-MM-DD/`
- ファイル: `00_agenda.md`, `01_context.md`, `02_decisions_actions.md` のように連番
- 内容: 目的 / 今日決めること / 主要論点 / 結論（決定）/ 次アクション（担当・期限）を最小セットで
