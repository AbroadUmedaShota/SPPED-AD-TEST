### 概要
ダッシュボード全体の多言語対応（i18n）を推進するため、共通のメッセージ管理基盤を導入し、アンケート回答画面、作成画面、サンクス画面の設定および表示を多言語化します。

### 目的
- グローバルユーザーへの対応を強化するため、UIの多言語表示を可能にする。
- メッセージの一元管理により、メッセージの保守性を向上させる。

### 主な変更内容
- `02_dashboard/src/services/i18n/messages.js` の新規作成（多言語メッセージ定義とユーティリティ関数の導入）
- アンケート回答画面 (`survey-answer.js`) の多言語対応
- アンケート作成画面 (`surveyCreation.js`) の多言語対応
- サンクス画面 (`thankYouScreen.js`, `thankYouScreenSettings.js`) の多言語対応と連続回答機能の追加
- UI共通コンポーネント (`surveyRenderer.js`) への翻訳リファレンス表示機能の追加
- 関連する HTML/CSS の更新

- [ ] 各画面の UI テキストがロケールに応じて切り替わること
