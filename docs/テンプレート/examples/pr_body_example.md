### 概要
Speed Review画面の「Active Question」等から開く設問選択モーダルにおいて、以下の改善を行いました。

1.  **設問番号の追加**: 
    -   **モーダル内**: 各設問の先頭に「Q1. 」「Q2. 」といった番号を付与しました。
    -   **ヘッダー**: 「Active Question」などのヘッダー部分にも、選択された設問の番号（「Qx. 」）が表示されるように修正しました。
2.  **グラフ化可否の視覚化（改善）**:
    -   **グラフ化可能**: `analytics` アイコン（青色）。
    -   **グラフ化不可**: 
        -   基本: `description` アイコン（文書/テキスト、グレー）。グラフアイコンと明確に区別しました。
        -   日時: `event` アイコン。
        -   表形式: `grid_on` アイコン。
        -   数値: `123` アイコン。
    -   ツールチップは削除し、アイコンの形状と色で直感的に判別できるようにしました。

### 変更内容
- `02_dashboard/src/speed-review.js`: 
    -   `populateQuestionSelector`: アイコン選定ロジックを刷新。
    -   `handleQuestionSelectClick`: ヘッダー更新ロジックの整理とコメントのクリーンアップ。

### 動作確認手順
1.  Speed Review画面を開く。
2.  「Active Question」ボタンをクリックしてモーダルを開く。
3.  設問リストに「Qx.」が付いていることを確認。
4.  グラフ化可能な設問は「グラフアイコン（青）」、そうでない設問は「文書アイコン（グレー）」等で明確に区別されていることを確認。
5.  設問を選択すると、ヘッダーにも「Qx.」付きで表示されることを確認。
