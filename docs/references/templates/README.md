# Issue / PR テンプレート（正本）

このディレクトリは、Issue・PR 作成およびレビューコメントの正規テンプレートを管理します。

## 収録ファイル
- `issue_body.md`: Issue 起票テンプレート。
- `issue_comment_body.md`: 実装提案コメントテンプレート。
- `pr_body.md`: Pull Request 本文テンプレート。
- `pr_comment_body.md`: セルフレビューコメントテンプレート。

## 運用ルール
- テンプレートを更新した場合は `docs/CONTRIBUTING.md` と関連スクリプトの参照パスを確認する。
- 過去案件ベースの記載例は `docs/templates/examples/` に置き、ここには置かない。
