# 管理者画面E2E実行チェックリスト（stg実画面観測ベース）

更新日: 2026-05-29

## 使い方

このチェックリストは、`admin_screen_e2e_observed_requirements.md` のstg実画面観測ベースE2E候補を手動実行用に落とし込んだものです。起点は `https://stg.speed-ad.com/admin/login` とし、ログイン後に到達できる管理者画面だけを対象にします。

実行時は `証跡` にスクリーンショット、ブラウザ名、画面幅、コンソールログ有無、URLを記録し、`結果` に `OK` / `NG` / `対象外` を記入します。

## 実行前提

- ブラウザはChromeを基準とし、ユーザーの認証済みセッションを使用する。
- 実データを更新し得る保存、削除、送信、確定、スキップは、ユーザー許可がある場合を除き実行しない。
- 実データの氏名、会社名、メールアドレス、個別数値は証跡や結果欄へ転記しない。
- 画面上にない要件は補完しない。
