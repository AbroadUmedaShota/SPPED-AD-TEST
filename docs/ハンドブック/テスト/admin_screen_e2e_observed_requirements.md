# 管理者画面E2E項目候補（stg実画面観測ベース）

更新日: 2026-05-29

## 位置づけ

この資料は、管理者画面を絡めたE2Eテスト項目を立てるためのstg実画面観測ベース要件一覧です。正式な業務仕様ではなく、`https://stg.speed-ad.com/admin/login` からログイン後に到達できた画面のDOM、表示テキスト、画面遷移、コンソール結果だけを根拠にします。

実行時の結果記入には、別紙の [`admin_screen_e2e_execution_checklist.md`](admin_screen_e2e_execution_checklist.md) を使用します。

## 根拠範囲

含める根拠:

- 認証済みChromeで到達できたstg管理者画面URL。
- 画面上に表示された見出し、フォーム、ボタン、リンク、テーブル見出し、ページング。
- 実データ値を除いたDOM観測結果。
- ブラウザコンソールで表示阻害エラーがないこと。

含めない根拠:

- ローカル実装ファイル、仕様書、過去資料。
- stg画面上にない状態遷移、権限、締切、監査ログ整合性。
- 実データの氏名、会社名、メールアドレス、個別数値を要件根拠にすること。

制約:

- 保存、削除、送信、確定、スキップなど実データを更新し得る操作は、候補化する場合も確認画面またはキャンセル可能範囲までに留める。
- 画面上に導線はあるが遷移先へ入れない、または一覧へ戻される場合は未到達として扱う。

## 実装状態の定義

| 実装状態 | 意味 | E2Eで期待する範囲 |
| --- | --- | --- |
| stgで実操作確認済み | ログイン後のstgで画面表示または導線遷移を実際に確認した | 表示、導線、エラーなしを確認する |
| stg表示のみ確認 | 画面と操作部品は確認したが、データ更新を伴う操作は実行していない | 表示、入力可能性、キャンセル可能範囲まで確認する |
| stg導線のみ確認 | リンクやボタンの存在を確認したが、遷移先の業務完了までは確認していない | URL遷移、戻り、リダイレクトを確認する |
| stg未到達/権限なし | 導線は観測したが、遷移先画面を確認できなかった | 未到達理由を記録する |
