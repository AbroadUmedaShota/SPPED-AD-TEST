### acc006 — プランを解約・ダウングレードしたい／アカウントを退会したい
_カテゴリ: アカウント・セキュリティ_

**変更前**
```
ダッシュボードの「アカウント設定」＞「プラン管理」から、プランの解約・ダウングレードおよびアカウント退会の手続きが可能です。月額（月単位）契約の場合、次回更新日の前日までに手続きを完了いただくことで、当月末での解約となり、追加費用は発生しません。年間契約・キャンペーン契約など契約形態によっては、解約金が発生する場合がありますので、契約書面および利用規約をご確認ください。退会手続き後、一定期間は業務データのエクスポートが可能です。保存期間経過後、業務データから削除いたします（法定保存義務のあるもの（請求関連データ等）は所定期間経過後に削除）。
詳細は利用規約および特定商取引法に基づく表示をご確認ください。
```
**変更後**
```
プレミアムプランの解約は、プレミアムプラン画面の「プランを解約する」から手続きできます。解約すると自動更新が停止し、現在の利用期限まではプレミアム機能をご利用いただけます。期限の翌日以降はスタンダードプランへ移行し、保存期間の延長やCSV出力などのプレミアム機能はご利用いただけなくなります。月額（月単位）契約の場合、次回更新日の前日までに手続きを完了いただくことで、当月末での解約となり、追加費用は発生しません（日割りでの返金はありません）。年間契約・キャンペーン契約など契約形態によっては解約金が発生する場合がありますので、契約書面および利用規約をご確認ください。
```
