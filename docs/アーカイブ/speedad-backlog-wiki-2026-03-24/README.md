# SPEED AD Backlog Wiki Full Backup Archive

このディレクトリは、Backlog wiki `SPDAD/Home` 配下からプロトタイプ理解に有用なページを選別し、ページ分割で保存した旧資料アーカイブです。

## Policy

- 原文を優先して保存します。要約ではなく、各 wiki ページの本文を個別 Markdown として残します。
- 運用資格情報や汎用テンプレートなど、資料価値より保管リスクまたはノイズが大きいページは整理対象として除外しています。
- 添付画像や添付ファイルは `assets/<wikiId>/` 配下に保存し、本文内リンクはローカル相対パスへ置き換えます。
- 添付が取得できない場合は、本文は必ず保存し、ページ末尾に `未取得` 注記と添付メタデータを残します。
- 外部リンク先の再帰取得は行いません。保存対象は wiki 本文とそのページに直接紐づく添付物までです。

## Layout

- `pages/`: wiki 1 ページ = 1 Markdown
- `assets/`: wiki ページに紐づく画像・添付ファイル
- `spd-ad-backlog-wiki-legacy.md`: ページ一覧と保存状況の索引

## Naming

- ページファイル名: `NNN_<wikiId>_<sanitized-title>.md`
- 添付ファイル名: Backlog 上の元ファイル名をできるだけ維持し、競合時のみ wikiId を付与します
- 見出し構成: ページ先頭に共通ヘッダを置き、Backlog 元ページ名、wikiId、更新日時、添付状態を明示します

## Notes

- これは現行仕様の正本ではなく、プロトタイプ期の旧資料です。
- 仕様反映時は `docs/` 配下の現行資料と突き合わせてください。
- 添付実ファイルの取得には Backlog API キーまたは認証済みセッションが別途必要です。
