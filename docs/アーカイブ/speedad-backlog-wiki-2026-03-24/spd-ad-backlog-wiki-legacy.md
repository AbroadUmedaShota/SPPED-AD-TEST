# SPEED AD Backlog Wiki Index

## 保存ルール

- `pages/NN_<wikiId>_*.md` に採用した wiki ページ本文を保存します。
- 添付の実バイナリ取得は、Backlog API キーまたは認証済みセッションがない限り実行できません。
- 添付が未取得でも本文保存を優先し、各ページ末尾に `未取得 attachment` とメタデータを記録します。
- 資料価値の低い汎用テンプレート、接続情報、運用資格情報メモ、単発 SQL メモは整理対象として除外しています。
