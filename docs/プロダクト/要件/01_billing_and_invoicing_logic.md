# 請求・課金ロジック定義書

## 1. 料金体系

本システムの商用利用における料金体系を以下に定義する。

### 1.1. プレミアムプラン基本料金
*   **月額料金:** 10,000円（税別）
*   **特典:** 初月（契約月）は基本料金無料。

### 1.2. グループ追加アカウント料金（マルチユーザー利用）
グループ機能を利用する場合、所属人数に応じて以下の追加料金が発生する。
*   **2名まで（管理者含む）:** 無料。
*   **3名以上:** 1名追加につき、月額 300円（税別）のアカウント料が発生。
*   計算例：
    *   管理者＋メンバー1名（計2名）: 0円
    *   管理者＋メンバー2名（計3名）: 300円/月
    *   管理者＋メンバー4名（計5名）: 900円/月

### 1.3. 従量課金要素（利用実績に応じた課金）
以下の項目については、利用実績に応じた費用が発生する。

#### 1.3.1. 名刺データ化費用
展示会で取得した名刺のデータ化件数に応じた費用。単価は選択された「スピードプラン」によって異なる。
*   **通常プラン（6営業日）:** 50円/枚
*   **特急プラン（3営業日）:** 100円/枚
*   **超特急プラン（1営業日）:** 150円/枚
*   **オンデマンドプラン（当日/1時間目標）:** 200円/枚
*   **オプション（多言語対応）:** プレミアムプラン基本料金（月額10,000円）に含まれる（追加費用なし）
*   **最低請求ルール:** 実際の作業件数が、事前に設定した「想定件数」の50%に満たない場合は、想定件数の50%分の料金を最低料金として請求する。

#### 1.3.2. お礼メール送信費用
システム経由で送信されたお礼メールの総通数に応じた費用。
*   **無料枠:** 1つのアンケート（または月間単位）につき、最初の100通まで無料。
*   **超過料金:** 101通目以降、1通につき 1円（税別）。

## 2. 請求仕様

### 1.1. 請求単位と契約形態

- プレミアムプランの契約は、「個人アカウント」または「グループ」のいずれかの単位で行うことができます。
- この契約単位は、アカウントまたはグループに付与される「プレミアムフラグ」によってシステム的に管理されます。

### 1.2. 請求書の宛先と発行ルール

- **個人契約の場合:**
  - 請求書は、プレミアムフラグが付与された**個人アカウント**に対して発行されます。
- **グループ契約の場合:**
  - 請求書は、プレミアムフラグが付与された**グループ**に対して発行されます。
  - 請求書の送付先は、当該グループの管理者アカウントに設定された連絡先となります。

### 1.3. 機能の利用範囲と制約

- **個人契約の場合:**
  - プレミアム機能は、契約した個人のみが利用可能です。
  - そのユーザーが所属するグループの他のメンバーは、そのプレミアム機能を利用することはできません。
- **グループ契約の場合:**
  - プレミアム機能は、契約したグループに所属する全てのメンバーが利用可能です。
  - ただし、メンバー個人のアカウントとしてプレミアム機能を利用することはできません（利用はグループのコンテキストに限定されます）。

## 2. 請求サイクルと支払い

### 2.1. 請求サイクル
- **締め日:** 毎月末日
- **支払日:** 翌月末払い

### 2.2. 支払い方法
- 原則として銀行振込とする。

## 3. 料金計算の基本ルール

### 3.1. 契約期間と料金発生
- 料金の日割り計算は行わない。月の途中で契約または解約した場合でも、1ヶ月分の月額料金が発生する。

### 3.2. 返金ポリシー
- 契約期間の途中で解約した場合、残期間に対する料金の返金は行わない。

### 3.3. 従量課金における最低請求ルール
- データ化サービスなど、一部の従量課金サービスにおいては最低請求数量が設定される場合がある。
- 例：名刺データ化において、実際の作業件数が事前に定めた想定件数の50%に満たない場合は、想定件数の50%分の料金を請求する。
