## 2. 主要な免責事項と制限 (2026-02-20 更新)

以下の外部要因に起因するサービスの停止、データの不具合、遅延について、当サービスは一切の責任を負わないものとする。

*   **インフラ障害:** AWS (Amazon Web Services) 等のクラウドインフラストラクチャの障害。
*   **外部API障害:** Google OCR などの外部連携APIの停止、仕様変更、または解析精度の低下。
*   **通信環境:** ユーザー側の通信品質の低下、ネットワーク遅延。
*   **データ精度:** 人力入力およびAI解析における100%の正確性は保証せず、商用利用における最終確認はユーザーの責任とする。

## 3. 料金・支払いに関する合意

詳細は [請求・課金ロジック定義書](../要件/01_billing_and_invoicing_logic.md) を参照。

*   **後払い原則:** 毎月末締め、翌月末払い。
*   **返金不可:** サービスの性質上、支払い済みの料金および従量課金確定後の返金は行わない。
*   **追加アカウント:** グループ利用時、3名以上の登録がある場合は追加料金が発生する。
