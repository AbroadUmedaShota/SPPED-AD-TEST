#### 利用不可理由（表示条件一覧）

モーダル内のバッジ（`利用不可`）や、ダウンロードボタン押下時のガイダンスとして表示される文言です。

| 対象 | 表示条件（概要） | 表示文言 |
| :--- | :--- | :--- |
| 共通 | 対象アンケート情報が渡されていない（`survey == null`） | ダウンロードするアンケート情報がありません。 |
| 共通 | 想定外の状態（フォールバック） | 現在ダウンロードできません。 |
| 回答データ | 回答数が 0（`answerCount + realtimeAnswers <= 0`） | 回答がないためダウンロードできません。 |
| 名刺画像 | 名刺機能を利用しない（`bizcardEnabled == false`） | 名刺機能を利用していないためダウンロードできません。 |
| 名刺画像 | 名刺登録が 0（`bizcardRequest <= 0`） | 名刺画像が登録されていないためダウンロードできません。 |
| 名刺画像 | DL期限終了（`status == 会期終了（DL期限終了）`） | ダウンロード期限を過ぎたためダウンロードできません。 |
| 名刺入力データ | 名刺機能を利用しない（`bizcardEnabled == false`） | 名刺機能を利用していないためダウンロードできません。 |
| 名刺入力データ | 名刺登録が 0（`bizcardRequest <= 0`） | 名刺が登録されていないためダウンロードできません。 |
| 名刺入力データ | DL期限終了（`status == 会期終了（DL期限終了）`） | ダウンロード期限を過ぎたためダウンロードできません。 |
| 名刺入力データ | データ化中（`status == 会期終了（データ化中）`） | 名刺データがまだ準備できていません。しばらくしてから再度お試しください。 |
| 名刺入力データ | 会期前/会期中（`status == 会期前 or 会期中`） | 会期終了後に名刺データが準備されます。 |
| 名刺入力データ | データ化対象外（`status == 会期終了`） | 名刺データ化対象外のためダウンロードできません。 |
| 名刺入力データ | データ化完了件数が 0（`bizcardCompletionCount <= 0`） | 名刺データが存在しないためダウンロードできません。 |
| 名刺入力データ | 上記に該当せずDL不可（`isDownloadable == false`） | 名刺データがまだ準備できていません。しばらくしてから再度お試しください。 |
