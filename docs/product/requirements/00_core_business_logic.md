# コアビジネスロジック定義書

## 1. 請求仕様

### 1.1. 請求単位と契約形態

- プレミアムプランの契約は、「個人アカウント」または「グループ」のいずれかの単位で行うことができます。
- この契約単位は、アカウントまたはグループに付与される「プレミアムフラグ」によってシステム的に管理されます。

### 1.2. 請求書の宛先と発行ルール

- **個人契約の場合:**
  - 請求書は、プレミアムフラグが付与された**個人アカウント**に対して発行されます。
- **グループ契約の場合:**
  - 請求書は、プレミアムフラグが付与された**グループ**に対して発行されます。
  - 請求書の送付先は、当該グループの管理者アカウントに設定された連絡先となります。

### 1.3. 機能の利用範囲と制約

- **個人契約の場合:**
  - プレミアム機能は、契約した個人のみが利用可能です。
  - そのユーザーが所属するグループの他のメンバーは、そのプレミアム機能を利用することはできません。
- **グループ契約の場合:**
  - プレミアム機能は、契約したグループに所属する全てのメンバーが利用可能です。
  - ただし、メンバー個人のアカウントとしてプレミアム機能を利用することはできません（利用はグループのコンテキストに限定されます）。
