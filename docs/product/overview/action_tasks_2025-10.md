# 対応アクションのタスク化（2025-10版）

## 背景
- 最新の開発状況サマリーで言及された「アンケート作成・一覧の機能拡張」と「リポジトリ整理の課題」を、着手順に落とし込んだタスクリスト。
- 各タスクは「担当」「成果物」「完了条件」を明示し、ブラウザスモークテストで検収する。

## 検証フロー
- UI 変更タスク（FE-01〜FE-04）は `python -m http.server 8000` で `/02_dashboard` を開き、主要ページの 404 とコンソールエラーがないことを確認する。
- パス整理タスク（CL-01/CL-03）は `rg` で旧パスを検出しないことを持って完了とし、必要に応じて `npx serve .` で回帰確認を行う。
- ドキュメント更新タスク（FE-05, CL-04）はレビューコメントを反映し、リンク切れチェックを行う。
