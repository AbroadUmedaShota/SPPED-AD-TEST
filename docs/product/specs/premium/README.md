# プレミアム関連仕様

このディレクトリは、プレミアム登録・契約管理に関する仕様書の正本を管理します。

## 収録ドキュメント
- `premium_page_requirements.md`: プレミアム加入促進ページ要件。
- `premium_registration_requirements.md`: プレミアム登録画面要件。
- `premium_dashboard_requirements.md`: 契約状況ダッシュボード要件。
- `premium_cancellation_requirements.md`: 解約手続き/完了画面要件。

## 運用メモ
- 旧配置 `docs/requirements/premium_*_requirements.md` は legacy redirect として維持します。
- プレミアム機能の検討ドラフトは `docs/service-consideration/` を参照してください。
