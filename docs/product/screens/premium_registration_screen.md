# プレミアム登録画面 設計書

## 1. 概要

本ドキュメントは、プレミアム登録画面のUI/UXに関する詳細な設計を定義します。

## 2. デザインコンセプト

- **全体:** モダンでクリーンなデザインとし、ユーザーがストレスなく登録プロセスを完了できるよう、直感的な操作性を提供します。
- **配色:** プロジェクトのブランドカラーを基調とし、アクセントカラーを効果的に使用して視覚的な誘導を行います。
- **フォント:** 可読性の高いフォント（例: Noto Sans JP, ヒラギノ角ゴ ProN）を使用します。
- **余白:** 十分な余白を確保し、圧迫感のないレイアウトを心掛けます。

---

## 3. 画面設計詳細

### 3.1. ステップ1: 基本情報入力

*   **画面タイトル:** `プレミアム登録 - 基本情報`
*   **ステップ表示:**
    *   `[1. 基本情報] -> 2. 会社情報 -> 3. 支払い情報 -> 4. 確認 -> 5. 完了`
    *   現在のステップ（1. 基本情報）をハイライト表示します。

#### 3.1.1. 画面要素

| 要素名           | 種類           | 説明                                                                                                                                     |
| ---------------- | -------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **氏名 (姓名)**  | テキスト入力   | 姓と名をそれぞれ入力。無料アカウント情報から自動引き継ぎ。必須項目。                                                                   |
| **電話番号**     | テキスト入力   | ハイフンあり/なしどちらでも可。無料アカウント情報から自動引き継ぎ。必須項目。                                                          |
| **部署名**       | テキスト入力   | 無料アカウント情報から自動引き継ぎ。任意項目。                                                                                         |
| **役職名**       | テキスト入力   | 無料アカウント情報から自動引き継ぎ。任意項目。                                                                                         |
| **「戻る」ボタン** | ボタン         | （このステップでは表示しない）                                                                                                         |
| **「次へ」ボタン** | ボタン         | `次へ進む`。入力内容を検証し、問題なければ「ステップ2: 会社情報・住所情報入力」へ遷移。                                                  |
| **キャンセル**   | テキストリンク | 登録プロセスを中断し、元の画面（例: ダッシュボード）へ戻る。                                                                           |

#### 3.1.2. バリデーションルール

| 項目名         | ルール                                                               | エラーメッセージ                                               |
| -------------- | -------------------------------------------------------------------- | -------------------------------------------------------------- |
| **氏名 (姓名)**  | 姓・名それぞれ必須。                                                 | `姓と名を入力してください。`                                   |
| **電話番号**     | 必須。半角数字とハイフンのみ。10桁または11桁。                       | `電話番号の形式が正しくありません。`                           |

### 3.2. ステップ2: 会社情報・住所情報入力

*   **画面タイトル:** `プレミアム登録 - 会社・住所情報`
*   **ステップ表示:**
    *   `1. 基本情報 -> [2. 会社情報] -> 3. 支払い情報 -> 4. 確認 -> 5. 完了`
    *   現在のステップ（2. 会社情報）をハイライト表示します。

#### 3.2.1. 画面要素

| 要素名           | 種類           | 説明                                                                                                                                     |
| ---------------- | -------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **会社名**       | テキスト入力   | 無料アカウント情報から自動引き継ぎ。必須項目。                                                                                         |
| **郵便番号**     | テキスト入力   | 住所の自動入力機能付き。無料アカウント情報から自動引き継ぎ。必須項目。                                                                  |
| **住所**         | テキスト入力   | 都道府県、市区町村、番地などを入力。無料アカウント情報から自動引き継ぎ。必須項目。                                                       |
| **建物名・部屋番号** | テキスト入力   | 無料アカウント情報から自動引き継ぎ。任意項目。                                                                                         |
| **「戻る」ボタン** | ボタン         | `戻る`。入力内容を保持したまま「ステップ1: 基本情報入力」へ遷移。                                                                      |
| **「次へ」ボタン** | ボタン         | `次へ進む`。入力内容を検証し、問題なければ「ステップ3: 支払い情報確認・同意」へ遷移。                                                     |
| **キャンセル**   | テキストリンク | 登録プロセスを中断し、元の画面（例: ダッシュボード）へ戻る。                                                                           |

#### 3.2.2. バリデーションルール

| 項目名     | ルール         | エラーメッセージ               |
| ---------- | -------------- | ------------------------------ |
| **会社名**   | 必須。         | `会社名を入力してください。`   |
| **郵便番号** | 必須。半角数字7桁。 | `郵便番号の形式が正しくありません。` |
| **住所**     | 必須。         | `住所を入力してください。`     |

| 要素名                           | 種類         | 説明                                                                                                                                                                       |

| **「次へ」ボタン**                 | ボタン       | `次へ進む`。入力内容を検証し、問題なければ「ステップ4: 入力内容確認」へ遷移。                                                                                                 |

| 項目名                         | ルール | エラーメッセージ                                                   |
| ------------------------------ | ------ | ------------------------------------------------------------------ |
| **有料コンテンツ利用規約 同意**  | 必須。 | `利用規約および特定商取引法に基づく表記に同意してください。`     |

### 3.4. ステップ4: 入力内容確認

*   **画面タイトル:** `プレミアム登録 - 入力内容確認`
*   **ステップ表示:**
    *   `1. 基本情報 -> 2. 会社情報 -> 3. 支払い情報 -> [4. 確認] -> 5. 完了`
    *   現在のステップ（4. 確認）をハイライト表示します。

#### 3.4.1. 画面要素

| 要素名           | 種類         | 説明                                                                                                                                                                                   |
| ---------------- | ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **基本情報**     | テキスト表示 | ステップ1で入力した氏名、電話番号、部署名、役職名を表示。                                                                                                                                  |
| **会社情報・住所情報** | テキスト表示 | ステップ2で入力した会社名、郵便番号、住所、建物名などを表示。                                                                                                                            |
| **支払い情報**   | テキスト表示 | 「料金は月末締め、後払いとなります。」などの支払い情報を再度表示。                                                                                                                        |
| **同意事項**     | テキスト表示 | 「有料コンテンツ利用規約 および 特定商取引法に基づく表記に同意済み」であることがわかるように表示。                                                                                        |
| **編集ボタン**   | ボタン/リンク  | 各項目の横に配置し、クリックすると該当の入力ステップ（例: ステップ1）に戻る。                                                                                                              |
| **「戻る」ボタン** | ボタン       | `戻る`。入力内容を保持したまま「ステップ3: 支払い情報確認・同意」へ遷移。                                                                                                                   |
| **「登録する」ボタン** | ボタン       | `この内容で登録する`。クリックすると、プレミアム登録処理が実行され、成功すれば「ステップ5: 登録完了」へ遷移。処理中はローディング表示を行う。 |
| **キャンセル**   | テキストリンク | 登録プロセスを中断し、元の画面（例: ダッシュボード）へ戻る。                                                                                                                           |

### 3.5. ステップ5: 登録完了

*   **画面タイトル:** `プレミアム登録完了`
*   **ステップ表示:**
    *   `1. 基本情報 -> 2. 会社情報 -> 3. 支払い情報 -> 4. 確認 -> [5. 完了]`
    *   現在のステップ（5. 完了）をハイライト表示します。

#### 3.5.1. 画面要素

| 要素名                   | 種類         | 説明                                                                                                                                                         |
| ------------------------ | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **完了メッセージ**       | テキスト表示 | 「プレミアム登録が完了しました。」といった成功メッセージを表示。                                                                                            |
| **今後の流れ案内**       | テキスト表示 | 「ご登録のメールアドレス宛に登録完了通知をお送りしました。請求書は翌月月初に発行されますので、ダッシュボードからご確認ください。」といった案内文を表示。 |
| **「ダッシュボードへ」ボタン** | ボタン       | `ダッシュボードへ進む`。クリックすると、プレミアム機能が利用可能になったダッシュボードへ遷移。                                                         |
