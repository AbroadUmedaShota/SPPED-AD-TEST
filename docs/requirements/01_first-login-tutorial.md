# 初回ログインチュートリアル要件

## 1. 目的

新規アカウント作成者がSPEED ADの基本的な使用方法を習得するためのトレーニング機能を提供する。ユーザーは指示に従って画面上の特定の部分を操作し、主要な機能（特にアンケート作成）の流れを体験する。

## 2. 対象画面

- チュートリアル開始点: `/04_first-login/index.html`
- チュートリアル内で遷移する画面:
    - `/02_dashboard/index.html` (ダッシュボード)
    - `/02_dashboard/surveyCreation.html` (アンケート作成画面)

## 3. 機能要件

### 3.1. チュートリアル体験
- **自動操作とユーザー操作の組み合わせ:**
    - 基本的にはシステムが自動で画面を操作し、テキスト入力や選択を行う。
    - ユーザーには「次へ」ボタンのクリックなど、特定の操作のみを許可する。
- **フォーカスとオーバーレイ:**
    - 現在操作対象のUI要素をハイライト（フォーカス）する。
    - フォーカスされている要素以外は、オーバーレイで覆いクリックできないようにする。
- **非永続化:**
    - チュートリアル中に作成されるアンケートなどのデータは、サーバーに保存されない。

### 3.2. 制御機能
- **チュートリアルのスキップ:**
    - ユーザーはチュートリアル全体をスキップできる。
- **チュートリアルの再実行:**
    - サイドバーなどに、チュートリアルを再度実行するためのボタンを設置する。（設置場所は別途検討）

## 4. トレーニング内容

### 4.1. 必須トレーニング: アンケート作成

#### シナリオ
新規アンケートの作成から設問の配置、プレビューまでの一連の流れをトレーニングする。

#### ステップ
1.  **ダッシュボード (`/02_dashboard/index.html`)**
    1.  「アンケート新規作成」ボタンをフォーカスする。
    2.  ユーザーがクリックすると、アンケート作成モーダルが表示される。

2.  **アンケート作成モーダル**
    1.  「アンケート名」入力欄をフォーカスし、サンプル名（例: `初めてのアンケート`）を自動入力する。
    2.  「表示タイトル」入力欄をフォーカスし、サンプルタイトル（例: `製品Aに関する満足度調査`）を自動入力する。
    3.  「回答期間」の入力ボックスとカレンダーをフォーカスし、本日から3日後までを自動で選択する。
    4.  「作成する」ボタンをフォーカスする。
    5.  ユーザーがクリックすると、アンケート作成画面 (`/02_dashboard/surveyCreation.html`) に遷移する。

3.  **アンケート作成画面 (`/02_dashboard/surveyCreation.html`)**
    1.  画面上部の「アンケート名」「表示タイトル」「アンケート期間」を順にフォーカスし、前のステップで入力した内容が反映されていることを示す。
    2.  「名刺データ設定」「お礼メール設定」「サンクス画面設定」の各項目を順にフォーカスし、簡単な機能説明をツールチップなどで表示する（画面遷移はしない）。
    3.  「設問グループ」エリアをフォーカスし、自動クリックで新しいグループを作成する。
    4.  画面右下の「+」（設問追加）ボタンをフォーカスする。
    5.  自動クリックで設問を追加し、以下の3つの設問を自動で作成・配置する。
        - シングル選択
        - マルチ選択
        - ドロップダウン
    6.  「プレビュー」ボタンをフォーカスし、自動クリックでプレビューモーダルを表示する。
    7.  プレビューモーダル全体をフォーカスし、内容を確認させる。
    8.  「アンケートを保存」ボタンをフォーカスする。
    9.  ユーザーがクリックすると、「アンケート作成のトレーニングは以上です。」といったメッセージを表示し、チュートリアルを終了する。
