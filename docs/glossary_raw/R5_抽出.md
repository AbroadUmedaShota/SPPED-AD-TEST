## 気づいた表記ゆれ・同音異義

### 利用者側アカウントの呼称

- **HTML（サイドバー）**: `ユーザー管理`
- **HTML（画面タイトル）**: `利用者アカウント管理`
- **HTML（詳細ページ）**: `ユーザー ID`（項目ラベル）、`アカウントID`（テーブル列見出し）

- 同じ対象に「ユーザー」「利用者」「顧客」の3つが混在している。

### アンケートのステータス値

- `草稿`（HTML: survey-management.html select option、badge）
- `下書き`（仕様書: 20_admin_survey_management.md）
- → 同一状態に対して2表記が存在する。

- `実施中`（HTML: survey-management.html badge、survey-detail.html badge）
- `公開中`（HTML: user-detail.html badge, 仕様書: 20_admin_survey_management.md）
- `公開中`（HTML: survey-management.html の稼働サマリバッジの aria-label 系テキストには「実施中」、仕様書では「公開中」）

### 「所属」の意味の複数用途

- operator-management.html では `所属` = オペレーターが属するグループ（`所属グループ`との関係が曖昧）
- user-management.html では `部署・役職` = 利用者の組織上の所属
- → 「所属」という語が「オペレーターの所属グループ」と「利用者の組織上の部署」の2つの意味で使われている。
