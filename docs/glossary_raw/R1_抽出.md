## 気づいた表記ゆれ・同音異義

- 【期間フィールドの呼称が3種】同じ「アンケートの開催期間」を指す概念に対し、`index.html` のテーブルヘッダーと `surveyDetailsModal.html` は「展示会会期」、`surveyCreation.html` のフィールドラベルは「開催期間」、`newSurveyModal.html` / `duplicateSurveyModal.html` では「回答期間」と3種類の異なる表記が使われている。

- 【お礼メール vs 御礼メール】`surveyDetailsModal.html` の同一モーダル内で、セクション見出し・リンクラベル・ボタンは「お礼メール設定」を使用しているが、設定状況を表示するフィールドラベルのみ「御礼メール設定」（御礼）と異なる表記になっている。`surveyCreation.html` のリンクラベルも「お礼メール設定」。

- 【マトリクス/マトリックスの表記揺れが4種】同じ設問タイプに対して、リライト版仕様書04は「マトリックス(SA)/(MA)」（ックス・カッコ内英字）、リライト版仕様書12は「マトリクス SA/MA」（クス・スペース区切り）、`matrix-comparison.html` は「マトリクス設問（SA）」（クス・全角括弧）、旧仕様書01は「マトリクス（単一/複数）」（クス・日本語）と4種の表記が混在している。

- 【設問タイプ名の新旧体系差】リライト版仕様書（04）で定義された11種の設問タイプ名（フリーアンサー / シングルアンサー / マルチアンサー / ドロップダウン回答 / 評定尺度 / 数値回答 / 手書きスペース / 説明カード等）と、旧仕様書（01）の設問タイプ名（自由記述 / 単一選択 / 複数選択 / ドロップダウン / 数値 / 手書き / 説明等）が完全に別体系で存在し、グラフ分析仕様書12では「単一選択 / 複数選択 / 評価尺度」と第三の混在表記が使われている。

- 【SPEEDレビューのスペース有無】`speed-review.html` H1、仕様書 `11_speed_review.md`、`06_speed_review.md` は「SPEEDレビュー」（スペースなし）で統一されているが、ブラウザタブタイトルは「SPEED-AD - SPEEDレビュー」と製品名部分は「-」区切り、画面名部分はスペースなしと混在している。

- 【デバイス切替ボタンのラベル差】`surveyPreviewModal.html`（v1）は「スマートフォン表示」「タブレット表示」、`surveyPreviewModalV2.html`（v2）は「スマートフォン」「タブレット」と「表示」の有無で異なる。

- 【フィルターオプションの粒度差】SPEEDレビューのデータ化ステータスフィルターについて、`speed-review.html` は「データ化完了」「データ化進行中」「全て表示」、`11_speed_review.md` は「全て / 完了 / 進行中」と表記が異なる（「データ化」プレフィックスの有無）。
