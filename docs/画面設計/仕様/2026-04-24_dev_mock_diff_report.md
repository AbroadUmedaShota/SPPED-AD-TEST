# 開発環境・モックアップ画面 差分調査

## 調査対象と前提

- 対象は利用者画面、公開画面、法務・ヘルプ、回答画面、利用者側モーダル。管理者画面 `03_admin/` 相当は今回の主比較から除外。
- 開発環境: `https://dev.speed-ad.com`
- ローカルモック: `http://localhost:8000`
- 認証後ユーザー: `abroadcrew02@gmail.com`
- 比較ID: 開発環境で存在確認できた `441` と `445` を中心に確認。
- 保存、送信、招待メール送信、回答送信など、データを書き換える操作は未実施。
- ローカルモック側はHTML/JS/仕様書のコードベース確認も併用。開発環境側の保存後・送信後・トークン付きURL後の挙動は、この静的モックリポジトリにサーバー実装が無いため、実操作またはLaravel側コード確認が必要。
- 前回の見落としを修正し、`thanks_mail?id=441`、`user_admin`、`user_admin_add`、`groupplan`、`group_join` を開発環境の実在ルートとして反映した。
