# 管理者画面 現行モック参考メモ

## 1. 目的

現行モックを、正本要件から分離した参考情報として記録する。

この資料は、後続の差分確認や実装時の棚卸しに使う。ここに書いた内容は仕様上の正解ではない。

## 2. 対象外

- 正本要件の定義
- 既存HTML構成の追認
- 削除済み資料の復元
- UI再設計、HTML/JS改修

## 3. 判断根拠

| 種別 | 扱い |
| --- | --- |
| `01_`〜`70_` の管理者画面資料 | 正本 |
| `03_admin/` | 参考: 現行モック |
| `BY-*` | 参考: 業務特化画面 |
| `sample/` | 参考: 検証・サンプル |
| `Moved` HTML | 参考: 移行導線 |

## 5. 分離ルール

- 現行モックから得た情報は、必ず `参考: 現行モック` と明記する。
- 現行モックにある画面や操作を、そのまま必要要件とは扱わない。
- 正本資料に転記する場合は、管理対象、責務、利用者側接点のいずれに該当するかを確認する。
