# ツールチップ仕様定義書

本ドキュメントは、SPEED ADの各画面に配置されるヘルプアイコン（？）およびツールチップの文言と表示ロジックを定義する。
※「250123_SPEEDAD_ツールチップ一覧.xlsx」および「2026-02-20 企画検討会議 議事録」の内容を統合。
※アンケート作成・編集画面、お礼メール設定画面、名刺データ化設定画面については、モック側の最新化内容を正本とし、開発中画面との差分が残っている場合でも本書を開発中画面に合わせて巻き戻さない。

---

## 1. アンケート一覧画面

| 項目 | 配置場所 | ツールチップ文言（表示内容） |
| :--- | :--- | :--- |
| **アクション** | 「アクション」の右側 | **アクション**<br>**編集ボタン**: アンケートの編集ができます。会期開始後は編集できなくなります。<br>**QRボタン**: アンケート回答画面のQR表示、ダウンロード、URLコピーができます。<br>**SPEEDレビュー**: リアルタイムでアンケートの回答状況や名刺画像を確認できます。<br>**データダウンロード**: 回答結果や名刺画像、名刺データのダウンロード画面を開きます。 |
| **アンケート名** | 「アンケート名」の右側 | **アンケート名**: 御社内でアンケートを識別するための名前です。回答者の画面では表示されません。 |
| **展示会会期** | 「展示会会期」の右側 | **展示会会期**: アンケートの有効回答期間です。この期間前に登録された回答はサンプルとしてデータ格納されます。この期間後はアンケートの回答が出来なくなります。開始日は設定・変更する日の翌日から、終了日は開始日と同日～30日後までの範囲で設定可能です。<br>会期開始後は、アンケート名、会期、設問内容、名刺データ化設定の変更ができなくなります。 |
| **表示タイトル** | アンケート詳細モーダル内「基本情報」の「表示タイトル」の右側 | **アンケートタイトル**: 回答画面の冒頭に表示されるタイトルです。 |
| **メモ** | アンケート詳細モーダル内「基本情報」の「メモ」の右側 | **メモ**: 御社内のメモ欄です。回答者の画面では表示されません。 |
| **ダウンロード期限** | アンケート詳細モーダル内「データ＆請求」の「ダウンロード期限」の右側 | **ダウンロード期限**: 名刺データ化完了日から30日後（データ化なしの場合は会期終了日から30日後）が、回答結果・名刺画像・名刺データのダウンロード期限となります。 |

## 2. アンケート作成画面

| 項目 | 配置場所 | ツールチップ文言（表示内容） |
| :--- | :--- | :--- |
| **アンケート名** | 「基本情報」の「アンケート名（管理用）」入力欄の右側、およびアンケート新規作成モーダル内「アンケート名」入力欄の右側 | 同上 |
| **表示タイトル** | 「基本情報」の「表示タイトル」入力欄の右側、およびアンケート新規作成モーダル内「表示タイトル」入力欄の右側 | **アンケートタイトル**: 回答画面の冒頭に表示されるタイトルです。 |
| **説明** | 「基本情報」の「説明」入力欄の右側 | **アンケート説明文**: 回答画面で、アンケートタイトルと設問1の間に表示される文章です。 |
| **アンケート期間 / 回答期間** | 「基本情報」の「アンケート期間」入力欄の右側、およびアンケート新規作成モーダル内「回答期間」入力欄の右側 | 同上 |
| **メモ** | 「メモ」入力欄の右側、およびアンケート新規作成モーダル内「メモ」入力欄の右側 | **メモ**: 御社内のメモ欄です。回答者の画面では表示されません。 |
| **手書きスペース** | 「設問設定」の設問形式プルダウン内「手書きスペース」の右側 | **回答形式「手書きスペース」について**: 回答欄内に手書きで書き込みできるスペースです。タッチペンやタブレットでの利用を推奨します。回答内容は画像データとして保存され、ダウンロード可能です（テキストデータ化は今後のプラン拡張で対応予定）。 |
| **名刺画像添付を有効にする** | 「連携設定」の「名刺画像添付を有効にする」の右側 | **名刺の撮影をしない**: BtoCイベント等で名刺撮影を依頼しない場合に選択します。この場合、名刺データ化や御礼メール機能は利用できません。 |

## 3. 名刺データ化設定画面

| 項目 | 配置場所 | ツールチップ文言（表示内容） |
| :--- | :--- | :--- |
| **アンケート名** | 表示されているアンケート名の右側 | **アンケート名**: 御社内でアンケートを識別するための名前です。回答者の画面では表示されません。 |
| **展示会会期** | 表示されている展示会会期の右側 | **展示会会期**: アンケートの有効回答期間です。この期間前に登録された回答はサンプルとしてデータ格納されます。この期間後はアンケートの回答が出来なくなります。開始日は設定・変更する日の翌日から、終了日は開始日と同日～30日後までの範囲で設定可能です。<br>会期開始後は、アンケート名、会期、設問内容、名刺データ化設定の変更ができなくなります。 |
| **1. データ化プランと見込み枚数** | 「1. データ化プランと見込み枚数」の右側 | **名刺データ化プラン**: 記載の単価はアンケート回答1件＝名刺1枚あたりのデータ化単価です。納期は会期終了日の翌営業日を1営業日目としてカウントします。ご依頼件数の偏りや想定件数との乖離、イレギュラー等により前後する可能性があります。 |
| **見込み枚数** | 「見込み枚数」の右側 | **想定件数**: 見込まれるアンケート回答者数を入力してください。 |
| **割引クーポン** | 「料金見積もりサマリー」内「割引クーポン」の右側 | **クーポンコード**: クーポンをお持ちの方は入力してください。 |
| **名刺のデータ化を行う** | 「名刺のデータ化を行う」の右側 | **名刺のデータ化は行わない**: アンケートで名刺画像を取得した場合も、全件に対してデータ化処理を行いません。 |

## 4. グループ管理画面（作成・編集）

| 項目 | 配置場所 | ツールチップ文言（表示内容） |
| :--- | :--- | :--- |
| **グループ** | サイドバー「グループ新規作成」を押下した際に表示されるモーダル内「グループ新規作成」の右側、およびグループ管理画面「グループ管理」の右側 | **グループ**: アンケートを複数名で編集できるグループを作成できます。作成者が管理者となります。グループ作成後、グループメンバー管理から他の人を招待してグループに加える事ができます。個人のページとグループのページはサイドバーから切り替える事ができます。グループには管理者を含めて2名まで無料で所属する事ができます。3人目からはひとりにつき月額100円（＋税）のアカウント料で加える事ができます。グループのアカウント料、また名刺データ化費用などグループ内で作成したアンケートに関する費用は、グループの管理者宛のご請求となります。 |

## 5. お礼メール設定画面

| 項目 | 配置場所 | ツールチップ文言（表示内容） |
| :--- | :--- | :--- |
| **アンケート名** | 項目名の右側 | **アンケート名**: 御社内でアンケートを識別するための名前です。回答者の画面では表示されません。 |
| **会期期間** | 「会期期間」の右側 | **展示会会期**: アンケートの有効回答期間です。この期間前に登録された回答はサンプルとしてデータ格納されます。この期間後はアンケートの回答が出来なくなります。開始日は、アンケート新規作成日の翌日から設定する事ができます。終了日は、開始日と同日～30日後までの日にちで設定する事ができます。<br>会期開始後は、以下の変更ができなくなります。<br>・アンケート名や展示会会期の変更。<br>・アンケートの設問の変更、またアンケートの削除。<br>・名刺データ化設定の変更。 |
| **送信しない** | 送信設定カードの「送信しない」ラベル右側 | **御礼メールを送らない**: 「送信しない」を選択すると、御礼メールの送信確認メールが届かなくなり、御礼メールは送信されません。 |

## 6. 実装上の注意
- ヘルプアイコンは Material Icons の `help_outline` を使用する。
- ツールチップの表示には、既存の CSS フレームワークまたは `tippy.js` 等の導入を検討する（現在は標準の `title` 属性またはカスタムのホバー表示）。
