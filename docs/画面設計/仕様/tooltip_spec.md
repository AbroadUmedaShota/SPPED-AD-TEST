# ツールチップ仕様定義書

本ドキュメントは、SPEED ADの各画面に配置されるヘルプアイコン（？）およびツールチップの文言と表示ロジックを定義する。
※「250123_SPEEDAD_ツールチップ一覧.xlsx」および「2026-02-20 企画検討会議 議事録」の内容を統合。

---

## 1. アンケート一覧画面

| 項目 | 配置場所 | ツールチップ文言（表示内容） |
| :--- | :--- | :--- |
| **操作ボタン説明** | 編集／QR／複製／削除 ボタンの上 | **編集ボタン**: アンケートの編集ができます。なおアンケート会期の開始日以降は、編集したアンケートを保存できなくなります。<br>**QRボタン**: アンケート回答画面のQR画像の表示・ダウンロード、またURLのコピーができます。<br>**複製ボタン**: アンケートのコピーを作成できます。過去のアンケートの設問を流用したい場合に活用いただけます。<br>**削除ボタン**: アンケートを削除できます。なおアンケート会期の開始日以降は、アンケートの削除ができなくなります。 |
| **アンケート名** | 「アンケート名」の右側 | **アンケート名**: 御社内でアンケートを識別するための名前です。回答者の画面では表示されません。 |
| **展示会会期** | 「展示会会期」の右側 | **展示会会期**: アンケートの有効回答期間です。この期間前に登録された回答はサンプルとしてデータ格納されます。この期間後はアンケートの回答が出来なくなります。開始日は設定・変更する日の翌日から、終了日は開始日と同日～30日後までの範囲で設定可能です。<br>会期開始後は、アンケート名、会期、設問内容、名刺データ化設定の変更ができなくなります。 |
| **ダウンロード期限** | 「ダウンロード期限」の右側 | **ダウンロード期限**: 名刺データ化完了日から30日後（データ化なしの場合は会期終了日から30日後）が、回答結果・名刺画像・名刺データのダウンロード期限となります。 |

## 2. アンケート作成画面

| 項目 | 配置場所 | ツールチップ文言（表示内容） |
| :--- | :--- | :--- |
| **アンケート名** | 入力欄の右側 | 同上 |
| **アンケートタイトル** | 入力欄の右側 | **アンケートタイトル**: 回答画面の冒頭に表示されるタイトルです。 |
| **アンケート説明文** | 入力欄の右側 | **アンケート説明文**: 回答画面で、アンケートタイトルと設問1の間に表示される文章です。 |
| **展示会会期** | 入力欄の右側 | 同上 |
| **メモ** | 入力欄の右側 | **メモ**: 御社内のメモ欄です。回答者の画面では表示されません。 |
| **手書きスペース** | 設問形式「手書き」の右側 | **手書きスペースについて**: 回答欄内に手書きで書き込みできるスペースです。タッチペンやタブレットでの利用を推奨します。回答内容は画像データとして保存され、ダウンロード可能です（テキストデータ化は今後のプラン拡張で対応予定）。 |
| **名刺の撮影をしない** | 選択肢の右側 | **名刺の撮影をしない**: BtoCイベント等で名刺撮影を依頼しない場合に選択します。この場合、名刺データ化や御礼メール機能は利用できません。 |

## 3. 名刺データ化設定画面

| 項目 | 配置場所 | ツールチップ文言（表示内容） |
| :--- | :--- | :--- |
| **項目・スピードプラン** | プラン選択肢の右側 | **項目プランとスピードプラン**: 記載の単価は回答1件あたりのデータ化単価です。無料プランは2項目（メール・氏名）、スタンダードは10項目をデータ化します。通常作業は6営業日、特急は3営業日を予定しています（会期終了日の翌営業日からカウント）。状況により前後する可能性があります。 |
| **想定件数** | 入力欄の右側 | **想定件数**: 見込まれるアンケート回答者数を入力してください。 |
| **クーポンコード** | 入力欄の右側 | **クーポンコード**: クーポンをお持ちの方は入力してください。 |
| **名刺のデータ化は行わない** | 選択肢の右側 | **名刺のデータ化は行わない**: アンケートで名刺画像を取得した場合も、全件に対してデータ化処理を行いません。 |

## 4. グループ管理画面（作成・編集）

| 項目 | 配置場所 | ツールチップ文言（表示内容） |
| :--- | :--- | :--- |
| **グループについて** | 「グループ」見出しの右側 | **グループ**: アンケートを複数名で編集できるグループを作成できます。作成者が管理者となります。個人のページとグループのページはサイト上部で切り替え可能です。 |
| **メンバー管理・料金** | メンバー管理セクション右側 | **メンバー管理**: メンバーの招待や権限管理を行います。ドラッグ&ドロップで表示順を変更できます。<br>**【料金について】**: グループには管理者を含めて2名まで無料で所属できます。3人目からはひとりにつき月額300円（税別）のアカウント料が発生します。グループに関する費用は管理者に請求されます。 |

---

## 5. 実装上の注意
- ヘルプアイコンは Material Icons の `help_outline` を使用する。
- ツールチップの表示には、既存の CSS フレームワークまたは `tippy.js` 等の導入を検討する（現在は標準の `title` 属性またはカスタムのホバー表示）。
- 料金に関する記述は、最新の議事録（2026-02-20）に基づき、300円/月（税別）を正とする。
