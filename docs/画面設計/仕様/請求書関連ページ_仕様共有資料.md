# 請求書関連ページ 早見表

## 1. 目的

- 利用者向け請求書ページの正本と補助資料を素早く辿るための入口です。
- 本書は詳細ルールを重複記載しません。仕様更新時は、まず正本を更新してください。
- 請求関連全体の分類は `docs/画面設計/仕様/請求関連仕様マップ.md` を参照してください。

## 2. 対象画面（利用者向け）

| 画面ID | パス | 役割 |
| :--- | :--- | :--- |
| INV-001 | `02_dashboard/invoiceList.html` | 請求書一覧 |
| INV-002 | `02_dashboard/invoice-detail.html` | 請求書詳細（Invoice Sheet 表示 + PDF出力） |

- 決済実行、督促、メール送信運用
- 社内向けの価格判断、請求ポリシー、会議経緯

## 4. URL と ID の必須ルール

### 4.1 URL

- `invoice-detail.html?id={id}`
- `invoice-print.html?id={id}`

### 4.2 `id`

`id` は以下のいずれかです。

- 元請求書 ID: `INV-YYYY-MM-SEQ`
- 集計 ID: `AGG-YYYYMM-{GROUP|PERSONAL}`（内部互換用）

### 4.3 一覧からの遷移

- INV-001 は請求書単位で、個人アカウント請求と各グループアカウント請求を同一時系列に表示します。
- 一覧から詳細への遷移は原則として `INV-...` を使います。

### 4.4 表示用請求書番号

- 画面上の請求書番号は `YY-ユーザーID5桁-連番3桁` 形式です。
- URL の `id` と表示用請求書番号は同一ではありません。

## 5. 実装再確認時の要点

- INV-001:
  - `issueDate` 降順で請求書カードを表示
  - 請求先アカウント名と請求先を識別表示する
  - カード押下と `Enter` / `Space` で INV-002 へ遷移
- INV-002:
  - `#invoice-sheet-container` を画面内表示し、`帳票ダウンロード` で PDF 化
  - 請求先は `accountId` から `users/groups` を引いて解決
- INV-003:
  - `id` に応じた印刷用 HTML を生成し、描画後に `window.print()` を自動実行

## 6. 実装参照

- 一覧データ整形・請求先表示名解決: `02_dashboard/src/invoiceList.js`
- ID 解決/明細連結/税計算: `02_dashboard/src/services/invoiceService.js`
- 請求先解決/PDF 生成: `02_dashboard/src/invoiceDetail.js`
- 印刷 HTML 生成: `02_dashboard/invoice-print.html`

## 7. 既知の差分と制約

- 改ページ、金額計算、帳票レイアウトは `05_invoice_document.md` を正本とします。
- `isSubtotal: true` の行は INV-002 の詳細/PDF では表示対象です。INV-003 の印刷差分は `05_invoice_document.md` を参照してください。
- 品目ラベルは具体課金名で固定し、`従量課金` は請求書の表示ラベルに使いません。品目1 / 品目2 の正本は `04_invoice_screen.md` と `05_invoice_document.md` を参照してください。
- 追加アカウント料金の `品目2` は `請求対象 Nアカウント` のように請求対象人数のみを表示します。無料アカウント数 `2` 固定や `3アカウント目以降` の課金ルールは内部条件として扱います。
- 利用者向けの主導線では `AGG-...` を使わず、内部互換としてのみ残します。
- `items[].category` と `items[].taxable` は帳票整合上の必須項目です。
- `02_dashboard/seikyuusyo_sample.html` はサンプル/参考実装であり、現行導線の正本ではありません。

## 8. 管理者側との境界

- 利用者向け請求書ページの正本は `docs/画面設計/仕様/` 側に置きます。
- 管理者向け請求管理は `docs/画面設計/仕様/admin/billing_management_requirements.md` を正本とし、責務を分離します。
