# surveyCreation.html 機能一覧

`docs/画面改修/` から `docs/画面設計/仕様/` に移管した補足資料です。survey creation 系の正本群の一部として扱います。

## 1. 基本情報セクション（アコーディオン）

| 機能 | 詳細 |
|------|------|
| 多言語機能の有効化トグル | アンケートごとに多言語入力を切り替えるON/OFFスイッチ（プレミアムプラン限定） |
| 追加言語の選択 | 多言語ON時に表示、利用言語を選択するパネル |
| 入力言語タブ切り替え | 多言語ON時に表示、言語ごとにテキストを入力するタブUI |
| アンケート名（管理用）入力 | 管理用のアンケート名テキスト入力（必須） |
| 表示タイトル入力 | 回答者に見えるタイトルテキスト入力（必須） |
| 説明入力 | アンケートの説明テキストエリア（任意） |
| アンケート期間設定 | 開始〜終了の日付範囲ピッカー（必須） |

## 2. 連携設定セクション

| 機能 | 詳細 |
|------|------|
| 名刺画像添付の有効化トグル | アンケート回答時に名刺撮影ステップを追加するON/OFFスイッチ |

## 3. 追加設定セクション

| 機能 | 詳細 |
|------|------|
| 名刺データ化設定ボタン | 別モーダルで名刺OCR設定を開く |
| お礼メール設定ボタン | 別モーダルでお礼メールを設定する |
| サンクス画面設定ボタン | 別モーダルでサンクス画面を設定する |
| クーポン適用状態の表示 | クーポンコード適用中の場合にインジケーターを表示（読み取り専用） |

## 4. メモセクション

| 機能 | 詳細 |
|------|------|
| 内部メモ入力 | 管理者のみが見る自由記述メモ |

## 5. 設問設定セクション

| 機能 | 詳細 |
|------|------|
| 設問グループの追加 | 設問を束ねるグループを追加するボタン |
| グループタイトル入力 | グループに名前をつける入力欄 |
| グループの複製 | グループ丸ごとコピー |
| グループの削除 | グループ丸ごと削除 |
| グループのアコーディオン開閉 | グループを折り畳む |
| グループ内への設問追加ボタン | グループの中に設問を追加 |
| 設問の並び替え（ドラッグ） | Sortable.jsによるドラッグ&ドロップ並び替え（グループ間・設問間） |

## 6. 設問カード（各設問）

| 機能 | 詳細 |
|------|------|
| 設問タイプの切り替え | ドロップダウンUIで設問タイプを変更 |
| 設問文の入力 | 設問テキストの入力（多言語対応） |
| 必須回答設定 | 設問ごとのチェックボックス |
| 設問の複製 | 設問を1件コピー |
| 設問の削除 | 設問を削除 |
| 設問のアコーディオン開閉 | 設問詳細を折り畳む |

## 7. 設問タイプ別の回答内容設定

| 設問タイプ | 設定内容 |
|-----------|----------|
| フリーアンサー | 最小文字数・最大文字数 |
| シングルアンサー | 選択肢の追加・削除・並び替え・一括入力 |
| マルチアンサー | 選択肢 + 最大選択可能数 |
| ドロップダウン回答 | 選択肢の追加・削除・並び替え・一括入力 |
| 数値回答 | 最小値・最大値・単位・ステップ |
| マトリックス(SA/MA) | 行・列の追加・削除 |
| 日付/時間 | 「日付を表示」「時間を表示」のON/OFF |
| 手書きスペース | 描画エリアの高さ（プリセット or カスタム） |
| 説明カード | 説明文テキストエリア |

## 8. 高度設定（設問ごと）

| 機能 | 詳細 |
|------|------|
| 条件分岐（ジャンプ先設定） | 回答結果に応じてジャンプする設問を指定（プレミアムプラン限定） |

## 9. FABメニュー（フローティングボタン）

| 機能 | 詳細 |
|------|------|
| 設問グループの追加 | 5と同等機能、画面右下の固定ボタンから操作 |
| 各設問タイプの追加 | フリーアンサー・シングルアンサー・マルチアンサー・数値回答・ドロップダウン・マトリックス(SA)・マトリックス(MA)・日付/時間・手書きスペース・説明カード |

## 10. アクションボタン（フッター）

| 機能 | 詳細 |
|------|------|
| プレビュー表示 | 回答画面のプレビューを表示 |
| QRコード | アンケートのQRコードを表示（保存前は無効） |
| キャンセル | 編集を破棄して戻る |
| アンケート保存 | 入力内容を保存（バリデーション通過後に有効化） |

## 11. サイドパネル（アウトラインマップ）

| 機能 | 詳細 |
|------|------|
| アウトラインマップ | 設問一覧を右側に固定表示、クリックで各セクションへジャンプ |
| サイドパネルの開閉 | トグルボタンで表示/非表示 |
| プレビュー・保存ボタン（サイドパネル内） | フッターのボタンと同等機能 |
