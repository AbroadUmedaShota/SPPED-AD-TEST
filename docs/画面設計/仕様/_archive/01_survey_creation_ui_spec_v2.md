# アンケート作成・編集ページ 詳細設計書 (V2: リデザイン＆機能完全維持版)

## 1. 概要
本ドキュメントは、`surveyCreation.html` のリデザインにおける具体的な実装仕様を定義する。2026-04-02 時点で、本V2 UI は `02_dashboard/surveyCreation.html` に正式反映し、`02_dashboard/surveyCreation-v2.html` は確認用URLとして残置する。
従来の「グループ機能」を廃止し、情報の優先順位に基づいた「Focused Builder」デザインを採用することで、機能追加によるUIの肥大化（ごちゃごちゃ感）を解消し、直感的な操作感を実現する。

**関連資料**:
*   [画面構成ワイヤーフレーム (02_survey_creation_wireframe.md)](02_survey_creation_wireframe.md)

## 2. デザインコンセプト： "Simplified & Focused"
単なる「グループ機能の削除」にとどまらず、以下のリデザインによって「ごちゃごちゃ感」を抜本的に解消する。

1.  **アコーディオン・ファースト**: 
    *   編集時以外の設問カードは1行のスリムなサマリー表示（Collapsed）に統一。縦方向の視覚的ノイズを劇的に削減し、アンケート全体の俯瞰性を高める。
2.  **オプション設定の「グリッド配置」**: 
    *   バラバラだった「名刺データ化」「お礼メール」「サンクス画面」のボタンを、一つのカード内にグリッド形式で集約。UI上の余計な見出し（「付加機能」等）を排除し、視覚的ノイズを最小化する。

---

## 3. デザイン・システム (UIガイドライン)
*   **カラー**: `primary` (#1D76DB), `surface` (#FFFFFF), `error` (#D73A4A)
*   **余白**: カード間 `gap-6`, カード内 `p-6`。
*   **共通ルール**: 他のダッシュボード画面（`index.html`等）と左右の余白、コンテナ幅 (`max-w-6xl`) を完全に統一する。

---

## 4. 画面構造と詳細レイアウト

### 4.1. 2カラム・グリッド構成 (プロジェクト標準準拠)
*   **メインエリア (8/12カラム)**: 左側に配置。基本設定、オプション設定（見出しなし）、設問リスト。
*   **サイドバーエリア (4/12カラム)**: 右側に `sticky top-24` で固定。アウトラインとアクションパネルを格納。

### 4.2. オプション設定カード (UI見出しなし)
*   基本設定カードの下に配置。カード自体のタイトル（Header）は表示せず、いきなり `grid grid-cols-1 sm:grid-cols-2 gap-4` でボタンを配置する。
*   各ボタンは「アイコン + テキスト + 状態バッジ（設定済/未設定）」で構成され、ボタン自体が機能を説明する。

### 4.3. 設問エディタ（フラット・質問リスト）
*   **Collapsed状態**: Q番号、タイプ、設問文（truncate）を横並びで表示。編集ボタンクリックで展開。
*   **Expanded状態**: 背景色を `bg-blue-50/30` にし、既存の全設定項目を表示。
