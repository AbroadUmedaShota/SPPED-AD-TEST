# run-test-process 引き継ぎ

## 目的

ログイン機能（ユーザー `/login` ＋ 管理者 `/admin/login`）の stg.speed-ad.com 向けテスト一式。

## 対象URL / 制約

- ユーザー: `https://stg.speed-ad.com/login` → 成功時 `/dashboard`（title「アンケート一覧」）
- 管理者: `https://stg.speed-ad.com/admin/login` → 成功時 `/admin/top`（DOM/遷移先は実装時inspectで確定）
- 実セレクタ（ユーザー側既知）: `#email` / `#password` / `button[type=submit]:has-text("ログイン")`
- 認証情報は環境変数のみ（`SPEEDAD_EMAIL`/`SPEEDAD_PASSWORD`/`SPEEDAD_BASE_URL`、管理者は別変数）。成果物・ログ・コミットに書かない。
- 副作用あり操作（送信/保存/削除/再設定メール/新規作成確定）は実行しない。
- 実行基盤: `.pwtest/`（gitignore済み）、`npx playwright test --project=staging`、Node は PATH 前置き必要。
